## Currency Converter

A Currency Converter App Which can convert indian currency into 8 other Currencies which includes

+ Dollar
+ Australian Dollar
+ Canadian Dollar
+ Bitcoin
+ Peso
+ Yen
+ Diram
+ Sterling
